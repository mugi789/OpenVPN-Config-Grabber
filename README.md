# How to Install and Usage
1. Open terminal and Copy
2. ```sudo apt install git python3```
3. ```pip install requests```
4. ```git clone https://github.com/mugi789/OpenVPN-Config-Grabber/```
5. ```cd OpenVPN-Config-Grabber```
6. ```python3 script.py```
